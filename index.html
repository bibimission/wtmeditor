How to use the Awesome WTM Editor ?

This tool goal is to help modders to easily create and update girl packs. It manages folders creation, .ini files and image converting.

The folders you want to edit have to be in the wtmeditor folder. It can't detect files in parent folders.

It does'nt allows event code management yet. It will come in future update.


Here's a detailed pack creation procedure :

  • Type the folder's name. If it exists, press Load, its contents will be loaded. If not, you can click Create and all the base folders will be generated.
  • Fill the GirlInfo Form with the desired information.
  • When a picture has a red outline, it means that the img format is not WEBP or WEBM. Click on it and the Awesome WTM Editor will convert it for you.
  • BEWARE, the gif to webm conversion takes a few seconds, don't click multiples times on a red gif to convert it because it will make the app crash.
  • The photoshoot tab allows you to manage alls the photos. Click on a photo to update its characteristics. The img will be renamed based on your choices. Sometimes, re-editing the same photo is a bit buggy. You can refresh the page if you see a strange behaviour
